# @ladamczyk/qoq-prettier — Agent Context

Prettier configuration template for QoQ projects.

## Usage

Reference it in `.prettierrc`:

```json
"@ladamczyk/qoq-prettier"
```

Or in `prettier.config.js`:

```js
export default '@ladamczyk/qoq-prettier';
```

## Config

```json
{
  "trailingComma": "es5",
  "printWidth": 100,
  "singleQuote": true
}
```
